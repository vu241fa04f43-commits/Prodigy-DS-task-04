# 📊 Task-04: Sentiment Analysis & Visualization

## 📌 Objective

Analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes towards specific topics or brands.

---

## 📂 Dataset

Dataset used in this project is available here:
https://github.com/Prodigy-InfoTech/data-science-datasets/tree/main/Task%204

The dataset contains social media text data labeled with sentiments:

* Positive 😊
* Negative 😡
* Neutral 😐

---

## ⚙️ Tools & Technologies

* Python 🐍
* Pandas
* Matplotlib
* Seaborn
* TextBlob

---

## 🧹 Data Preprocessing

* Removed null values
* Converted text to lowercase
* Cleaned unwanted characters and symbols
* Prepared text for sentiment analysis

---

## 🧠 Sentiment Analysis

* Used **TextBlob** to calculate polarity scores
* Classified text into:

  * Positive (polarity > 0)
  * Negative (polarity < 0)
  * Neutral (polarity = 0)

---

## 📊 Visualizations

The following visualizations were created:

* 📌 Sentiment distribution (bar chart)
* 📌 Sentiment percentage (pie chart)
* 📌 Topic-wise sentiment comparison

---

## 📈 Results & Insights

* Identified overall sentiment trends
* Observed how different topics influence public opinion
* Highlighted dominant sentiment categories
